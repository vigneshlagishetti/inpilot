# Sample Interview Questions

Test your AI Interview Assistant with these questions:

## General Questions

1. "What is the difference between let and var in JavaScript?"
2. "Explain what is React and why use it?"
3. "What is the difference between SQL and NoSQL databases?"
4. "Tell me about yourself as a software developer"
5. "What are your greatest strengths and weaknesses?"

## Technical Questions

### JavaScript
1. "Explain closures in JavaScript with an example"
2. "What is the event loop in JavaScript?"
3. "What are promises and how do they work?"
4. "Explain async await in JavaScript"
5. "What is the difference between == and === in JavaScript?"

### React
1. "What are React hooks and why were they introduced?"
2. "Explain the difference between state and props in React"
3. "What is the virtual DOM in React?"
4. "How does useEffect work in React?"
5. "What is prop drilling and how can you avoid it?"

### Data Structures & Algorithms
1. "Reverse a linked list"
2. "Find the maximum subarray sum"
3. "Check if a string is a valid palindrome"
4. "Implement a binary search algorithm"
5. "Find two numbers in an array that sum to a target"

### System Design
1. "How would you design a URL shortener like bit.ly?"
2. "Design a rate limiting system"
3. "How would you design Twitter's timeline feature?"
4. "Design a caching system"
5. "How would you design a file storage system like Dropbox?"

## Behavioral Questions

1. "Describe a challenging project you worked on"
2. "Tell me about a time you had a conflict with a team member"
3. "How do you handle tight deadlines?"
4. "Describe a time when you failed and what you learned"
5. "Why do you want to work for our company?"

## Code-Specific Questions

Try these to see both brute force and optimal solutions:

1. "Write a function to find all pairs in an array that sum to a target value"
2. "Implement a function to detect if a linked list has a cycle"
3. "Write a function to merge two sorted arrays"
4. "Implement a function to find the longest substring without repeating characters"
5. "Write a function to validate if a binary tree is a valid binary search tree"

## Tips for Testing

- Speak clearly and at a normal pace
- Wait 2 seconds after finishing your question
- Try different types of questions to see varied answer formats
- Code questions will show brute force and optimal approaches
- General questions will show direct answers and detailed explanations
